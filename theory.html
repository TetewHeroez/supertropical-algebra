What is Supertropical Algebra?

Supertropical algebra is an algebraic structure that extends tropical algebra by introducing ghost elements. It provides a framework for solving optimization problems and has applications in algebraic geometry, phylogenetics, and computer science. This page provides comprehensive mathematical background on supertropical algebra, based on the extended semiring tropical framework.

1. Supertropical Algebra

1.1 Extended Semiring Tropical

As discussed in max-plus algebra, the tropical algebra structure lacks an inverse element for the addition operation Γèò. This limitation is specifically addressed by introducing the extended semiring tropical structure, allowing us to solve systems of linear equations.

Definition 1.1 (Extended Semiring Tropical)

An extended semiring tropical is defined as \((T, \oplus, \otimes)\) where \(T = \mathbb{R} \cup \{-\infty\} \cup \mathbb{R}^\nu\), where \(\mathbb{R}\) is the set of all real numbers and \(\mathbb{R}^\nu = \{a^\nu \mid a \in \mathbb{R}\}\).

  • The neutral element for \(T\) is \(\varepsilon \stackrel{\text{def}}{=} -\infty\)

  • The unit element is \(e \stackrel{\text{def}}{=} 0\)

The set \(\mathbb{R}_{-\infty} = \mathbb{R} \cup \{-\infty\}\) represents the ideal of \(T\), called the ideal ghost.

Meanwhile, a function \(v : T \to \mathbb{R}_{-\infty}\) defined by \(v(x) \stackrel{\text{def}}{=} x, \forall x \in \mathbb{R}_{-\infty}\) represents tangible identity, and \(v(a) = a^\nu, \forall a \in \mathbb{R}\) where element \(a^\nu\) is called the element ghost or ghost of \(a\). The function \(v\) is called the ghost mapping.

Definition 1.2 (Extended Semiring Tropical Order)

Given an extended semiring tropical \(T\), a partial order relation \(\prec\) is defined on \(T\) as follows:

For any \(a, b \in \mathbb{R}\), \(a^\nu, b^\nu \in \mathbb{R}^\nu\) and \(x \in T\):

  1. \(-\infty \prec x, \forall x \in T \setminus \{-\infty\}\)

  2. For any real numbers \(a \prec b\), then \(a \prec a^\nu, a \prec b^\nu, a^\nu \prec b\), and \(a^\nu \prec b^\nu\)

  3. \(a \prec a^\nu\) for any \(a \in \mathbb{R}\)

Axiom 1.1 (Extended Semiring Tropical Operations)

Given an extended semiring tropical \(T\), the operations \(\oplus\) (max) and \(\otimes\) (addition) on \(T\) satisfy the following axioms.

For any \(a, b \in \mathbb{R}\), \(a^\nu, b^\nu \in \mathbb{R}^\nu\) and \(x, y \in T\):

  1. \(-\infty \oplus x = x \oplus -\infty = x, \forall x \in T\)

  2. \(x \oplus y = \max_{\prec} \{x, y\}\) provided \(x \neq y\)

  3. \(a \oplus a = a^\nu \oplus a^\nu = a \oplus a^\nu = a^\nu \oplus a = a^\nu\)

  4. \(-\infty \otimes x = x \otimes -\infty = -\infty\)

  5. \(a \otimes b = a + b\) (classical addition)

  6. \(a^\nu \otimes b = a \otimes b^\nu = a^\nu \otimes b^\nu = (a + b)^\nu\)

Example 1.1: Operations in extended semiring tropical \(T\):

  1. \(-\infty \oplus 4 = 4 \oplus -\infty = 4\)

  2. \(-2 \oplus 7 = \max_{\prec} \{-2, 7\} = 7\)

  3. \(8 \oplus 8 = 8^\nu \oplus 8^\nu = 8 \oplus 8^\nu = 8^\nu \oplus 8 = 8^\nu\)

  4. \(-\infty \otimes 3 = 3 \otimes -\infty = -\infty\)

  5. \(6 \otimes 9 = 6 + 9 = 15\)

  6. \(7^\nu \otimes 8 = 7 \otimes 8^\nu = 7^\nu \otimes 8^\nu = (7 + 8)^\nu = 15^\nu\)

2. Ghost Surpasses Relation

2.1 Definition

In supertropical semiring \(R\), for any \(a \in R\), then \(a \oplus a = -\infty\) only holds when \(a = -\infty\). However, for any \(a \in T\) then \(a \oplus a = a^\nu\) and for any \(x \in G\) then \(x \oplus x = x\). Therefore, a ghost surpasses relation in \(R\) will be introduced as follows.

Definition 2.1 (Ghost Surpasses Relation)

Given a supertropical semiring \(R\), a relation \(\vDash\) is called a ghost surpasses relation in \(R\) defined as:

\[x \vDash y \quad \text{if} \quad x = y \oplus z \quad \text{for some} \quad z \in \mathcal{G}_0\]

where \(\mathcal{G}_0\) is the set of ghost elements.

Example 2.1: Ghost surpasses relation in supertropical semiring \(R\):

  1. For \(9, 10^\nu \in R\), we have \(10^\nu \vDash 9\) because \(10^\nu = 9 \oplus 10^\nu\). Here, \(z = 10^\nu\) is a value in \(\mathcal{G}_0\) satisfying \(10^\nu = 9 \oplus 10^\nu\).

  2. For \(-5 \in R\), we have \(-5 \vDash -5\) because \(-5 = -5 \oplus \varepsilon\). Here, \(z = \varepsilon \in \mathcal{G}_0\) with \(z \neq \varepsilon\) satisfying \(-5 = -5 \oplus z\) is \(z \prec -5\). For example, \(z = -6^\nu\) and \(z = -7.5^\nu\). Notice that there are many values of \(z \in \mathcal{G}_0\) satisfying \(-5 = -5 \oplus z\) other than \(z \prec -5\).

  3. Meanwhile, for \(7, 6 \in R\), we have \(7 \nvDash 6\) because the value of \(z\) satisfying \(7 = 6 \oplus z\) is only \(z = 7 \notin \mathcal{G}_0\).

3. Supertropical Matrices

3.1 Matrix Operations

The set of all \(m \times n\) matrices over supertropical semiring is denoted as \(M_{m \times n}(R)\), where matrix elements are members of \(R\). When the dimension of a matrix is square, denoted as \(M_n(R)\), the binary operations ⊕ and ⊗ in \(R\) can be extended to matrix operations in \(M_n(R)\). Furthermore, the ghost surpasses relation in \(R\) can also be extended to matrices in \(M_n(R)\).

If \(A \vDash B\) then \(a_{i,j} \vDash b_{i,j}\) for any \(i\) and \(j\) in \(\underline{n}\).

Matrix Addition

For matrices \(A, B \in M_{m \times n}(R)\), the addition \(A \oplus B\) is defined as:

\[[A \oplus B]_{i,j} = a_{i,j} \oplus b_{i,j}\]

for \(i \in \underline{m}\) and \(j \in \underline{n}\).

Example 3.1: Given matrices

\[\begin{split}A = \begin{pmatrix} 2 & 1 \\ 1 & 3 \end{pmatrix}, \quad B = \begin{pmatrix} 5 & 4^\nu \\ 2 & 1 \end{pmatrix}\end{split}\]

Then:

\[\begin{split}A \oplus B = \begin{pmatrix} 2 \oplus 5 & 1 \oplus 4^\nu \\ 1 \oplus 2 & 3 \oplus 1 \end{pmatrix} = \begin{pmatrix} 5 & 4^\nu \\ 2 & 3 \end{pmatrix}\end{split}\]

Matrix Multiplication

For matrix \(A \in M_{m \times p}(R)\) and \(B \in M_{p \times n}(R)\), the multiplication \(A \otimes B\) is defined as:

\[[A \otimes B]_{i,j} = \bigoplus_{k=1}^{p} a_{i,k} \otimes b_{k,j}\]

for \(i \in \underline{m}\) and \(j \in \underline{n}\).

Example 3.2: Using the same matrices \(A\) and \(B\) from Example 3.1:

\[\begin{split}A \otimes B = \begin{pmatrix} (2 \otimes 5) \oplus (1 \otimes 2) & (2 \otimes 4^\nu) \oplus (1 \otimes 1) \\ (1 \otimes 5) \oplus (3 \otimes 2) & (1 \otimes 4^\nu) \oplus (3 \otimes 1) \end{pmatrix}\end{split}\]
\[\begin{split}= \begin{pmatrix} 7 \oplus 3 & 6^\nu \oplus 2 \\ 6 \oplus 5 & 5^\nu \oplus 4 \end{pmatrix} = \begin{pmatrix} 7 & 6^\nu \\ 6 & 5^\nu \end{pmatrix}\end{split}\]

3.2 Advanced Matrix Operations

Matrix Transpose

For any matrix \(A \in M_{m \times n}(R)\), its transpose \(A^T \in M_{n \times m}(R)\) is defined by:

\[[A^T]_{i,j} = a_{j,i}\]

where \(i \in \underline{n}\) and \(j \in \underline{m}\).

Example 3.3: For the matrix

\[\begin{split}A = \begin{pmatrix} 2 & 1 & 5^\nu \\ 3 & 4 & 1 \end{pmatrix}\end{split}\]

The transpose is:

\[\begin{split}A^T = \begin{pmatrix} 2 & 3 \\ 1 & 4 \\ 5^\nu & 1 \end{pmatrix}\end{split}\]

Matrix Power

For any square matrix \(A \in M_n(R)\) and positive integer \(k\), the power \(A^k\) is defined recursively:

\[A^1 = A, \quad A^k = A^{k-1} \otimes A\]

Example 3.4: For the matrix

\[\begin{split}A = \begin{pmatrix} 1 & 2 \\ 0 & 1 \end{pmatrix}\end{split}\]

We can compute:

\[\begin{split}A^2 = A \otimes A = \begin{pmatrix} (1 \otimes 1) \oplus (2 \otimes 0) & (1 \otimes 2) \oplus (2 \otimes 1) \\ (0 \otimes 1) \oplus (1 \otimes 0) & (0 \otimes 2) \oplus (1 \otimes 1) \end{pmatrix}\end{split}\]
\[\begin{split}= \begin{pmatrix} 1 \oplus (-\infty) & 3 \oplus 3 \\ (-\infty) \oplus (-\infty) & (-\infty) \oplus 1 \end{pmatrix} = \begin{pmatrix} 1 & 3 \\ -\infty & 1 \end{pmatrix}\end{split}\]

Identity Matrix

The identity matrix \(I_n \in M_n(R)\) is defined as:

\[\begin{split}[I_n]_{i,j} = \begin{cases} 0 & \text{if } i = j \\ -\infty & \text{if } i \neq j \end{cases}\end{split}\]

It satisfies \(A \otimes I_n = I_n \otimes A = A\) for any \(A \in M_n(R)\).

3.3 Permanent (Supertropical Determinant)

The permanent of a square matrix \(A \in M_n(R)\) is defined as:

\[\text{per}(A) = \bigoplus_{\sigma \in S_n} \bigotimes_{i=1}^{n} a_{i,\sigma(i)}\]

where \(S_n\) is the set of all permutations of \(\{1, 2, \ldots, n\}\).

Example 3.5: For a \(2 \times 2\) matrix

\[\begin{split}A = \begin{pmatrix} 3 & 1 \\ 2 & 4 \end{pmatrix}\end{split}\]

The permanent is:

\[\text{per}(A) = (3 \otimes 4) \oplus (1 \otimes 2) = 7 \oplus 3 = 7\]

For a \(3 \times 3\) matrix

\[\begin{split}B = \begin{pmatrix} 1 & 2 & 0 \\ 2 & 1 & 3 \\ 1 & 0 & 2 \end{pmatrix}\end{split}\]

We compute over all 6 permutations:

  1. \((1,2,3) \to 1 \otimes 1 \otimes 2 = 4\)

  2. \((1,3,2) \to 1 \otimes 3 \otimes 0 = 4\)

  3. \((2,1,3) \to 2 \otimes 2 \otimes 2 = 6\)

  4. \((2,3,1) \to 2 \otimes 3 \otimes 1 = 6\)

  5. \((3,1,2) \to 0 \otimes 2 \otimes 0 = 2\)

  6. \((3,2,1) \to 0 \otimes 1 \otimes 1 = 2\)

Therefore \(\text{per}(B) = 4 \oplus 4 \oplus 6 \oplus 6 \oplus 2 \oplus 2 = 6\).

3.4 Adjoint Matrix and Pseudo-Inverse

The adjoint matrix \(\text{adj}(A)\) of a matrix \(A \in M_n(R)\) is defined by:

\[[\text{adj}(A)]_{i,j} = \text{per}(M_{j,i})\]

where \(M_{j,i}\) is the \((n-1) \times (n-1)\) minor obtained by deleting row \(j\) and column \(i\) from \(A\).

The pseudo-inverse or weak inverse \(A^\nabla\) is defined as:

\[A^\nabla = \text{adj}(A) \otimes (\text{per}(A))^{-1}\]

where \((\text{per}(A))^{-1} = -\text{per}(A)\) in supertropical algebra.

Example 3.6: For the matrix

\[\begin{split}A = \begin{pmatrix} 3 & 1 \\ 2 & 4 \end{pmatrix}\end{split}\]

We have \(\text{per}(A) = 7\) (from Example 3.5).

The minors are:

  • \(M_{1,1} = (4)\), so \(\text{per}(M_{1,1}) = 4\)

  • \(M_{1,2} = (2)\), so \(\text{per}(M_{1,2}) = 2\)

  • \(M_{2,1} = (1)\), so \(\text{per}(M_{2,1}) = 1\)

  • \(M_{2,2} = (3)\), so \(\text{per}(M_{2,2}) = 3\)

Therefore:

\[\begin{split}\text{adj}(A) = \begin{pmatrix} 4 & 2 \\ 1 & 3 \end{pmatrix}\end{split}\]

The pseudo-inverse is:

\[\begin{split}A^\nabla = \text{adj}(A) \otimes (-7) = \begin{pmatrix} 4 + (-7) & 2 + (-7) \\ 1 + (-7) & 3 + (-7) \end{pmatrix} = \begin{pmatrix} -3 & -5 \\ -6 & -4 \end{pmatrix}\end{split}\]

4. Linear Systems over Supertropical Algebra

4.1 Homogeneous and Non-Homogeneous Systems

As in ordinary linear algebra, linear systems over supertropical algebra can be classified into homogeneous systems and non-homogeneous systems. In supertropical algebra, the ghost surpasses relation will be used instead of the equality relation \(=\).

A homogeneous linear system over supertropical algebra is stated as \(A \otimes x \vDash b\). Meanwhile, if \(b = \mathcal{E}\) where \(\mathcal{E}\) is a vector whose elements are all \(\varepsilon\), then the system \(A \otimes x \vDash \mathcal{E}\) is called a homogeneous system over supertropical algebra.

4.2 Solution Analysis

As has been explained in max-plus algebra theory, the max-plus algebra structure \(\mathbb{R}_{\max} = (\mathbb{R}_{\varepsilon}, \oplus, \otimes)\) lacks inverse elements for the \(\oplus\) operation. In other words, if \(a \in \mathbb{R}_{\varepsilon}\), there does not exist \(x \in \mathbb{R}_{\varepsilon}\) satisfying \(a \oplus x = \varepsilon\) if and only if \(a = \varepsilon\). This limitation makes it difficult to directly solve homogeneous linear systems \(A \otimes x = b\) in \(\mathbb{R}_{\max}\).

As a motivation from the discussion of homogeneous systems in \(\mathbb{R}_{\max}\), the solution for non-homogeneous linear systems \(A \otimes x \vDash b\) will be provided.

4.3 Cramer’s Rule

The solution to the system \(A \otimes x \vDash b\) can be found using Cramer’s rule in supertropical algebra:

\[x = \text{adj}(A) \otimes b \otimes (\text{per}(A))^{-1}\]

where:

  • \(\text{per}(A)\) is the permanent (supertropical determinant)

  • \(\text{adj}(A)\) is the adjoint matrix

  • \((\text{per}(A))^{-1} = -\text{per}(A)\) in supertropical algebra

The matrix is nonsingular (has unique solution) if \(\text{per}(A)\) is tangible (not ghost).

Implementation: The package provides the solve() method that implements this algorithm automatically.

Example 4.1: Solve the system \(A \otimes x \vDash b\) where

\[\begin{split}A = \begin{pmatrix} 2 & 3 \\ 1 & 4 \end{pmatrix}, \quad b = \begin{pmatrix} 8 \\ 7 \end{pmatrix}\end{split}\]

Using the solve() method or manual computation:

Step 1: Compute the permanent

\[\text{per}(A) = (2 \otimes 4) \oplus (3 \otimes 1) = 6 \oplus 4 = 6\]

Since \(\text{per}(A) = 6\) is tangible, the system has a unique solution.

Step 2: Compute the adjoint matrix

The minors are:

  • \(M_{1,1} = (4)\), so \(\text{per}(M_{1,1}) = 4\)

  • \(M_{1,2} = (1)\), so \(\text{per}(M_{1,2}) = 1\)

  • \(M_{2,1} = (3)\), so \(\text{per}(M_{2,1}) = 3\)

  • \(M_{2,2} = (2)\), so \(\text{per}(M_{2,2}) = 2\)

Therefore:

\[\begin{split}\text{adj}(A) = \begin{pmatrix} 4 & 1 \\ 3 & 2 \end{pmatrix}\end{split}\]

Step 3: Apply Cramer’s rule

\[\begin{split}x = \text{adj}(A) \otimes b \otimes (-\text{per}(A)) = \begin{pmatrix} 4 & 1 \\ 3 & 2 \end{pmatrix} \otimes \begin{pmatrix} 8 \\ 7 \end{pmatrix} \otimes (-6)\end{split}\]

First compute \(\text{adj}(A) \otimes b\):

\[\begin{split}\text{adj}(A) \otimes b = \begin{pmatrix} (4 \otimes 8) \oplus (1 \otimes 7) \\ (3 \otimes 8) \oplus (2 \otimes 7) \end{pmatrix} = \begin{pmatrix} 12 \oplus 8 \\ 11 \oplus 9 \end{pmatrix} = \begin{pmatrix} 12 \\ 11 \end{pmatrix}\end{split}\]

Then multiply by \(-6\):

\[\begin{split}x = \begin{pmatrix} 12 + (-6) \\ 11 + (-6) \end{pmatrix} = \begin{pmatrix} 6 \\ 5 \end{pmatrix}\end{split}\]

Step 4: Verify the solution

\[\begin{split}A \otimes x = \begin{pmatrix} 2 & 3 \\ 1 & 4 \end{pmatrix} \otimes \begin{pmatrix} 6 \\ 5 \end{pmatrix} = \begin{pmatrix} (2 \otimes 6) \oplus (3 \otimes 5) \\ (1 \otimes 6) \oplus (4 \otimes 5) \end{pmatrix}\end{split}\]
\[\begin{split}= \begin{pmatrix} 8 \oplus 8 \\ 7 \oplus 9 \end{pmatrix} = \begin{pmatrix} 8^\nu \\ 9 \end{pmatrix}\end{split}\]

We check: \(8^\nu \vDash 8\) ✓ and \(9 \vDash 7\) ✓ (since \(9 > 7\)).

Therefore \(x = \begin{pmatrix} 6 \\ 5 \end{pmatrix}\) is indeed a solution.

Implementation Notes

This package implements the supertropical semiring with:

  • Tangible elements: Standard numerical values

  • Ghost elements: Values marked with the is_ghost flag, displayed with ╬╜

  • Zero element: Represented as -math.inf

  • Efficient operations: Using NumPy arrays for matrix computations
